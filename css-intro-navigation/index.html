Create Navigational Links Within A Website

There are several techniques to navigate within a website. The semantic tag for navigation is the <nav> tag. You can navigate through your website by using relative addressing. For our examples we will use the unordered list tag(<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page(on page navigation)

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. HOWEVER, this header is optional. If you omit the header tag youu will recieve a documant outliner error: untitled section. Until you are shown how to avoid this error message, it will be acceptable on yourdocument outline.
